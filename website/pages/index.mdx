## 核心能力

| 功能 | 描述 |
|---|---|
| **语音转写** | 一键录音，自动生成带时间戳的文字笔记 |
| **AI 润色** | 将口语化转写内容整理为结构化笔记 |
| **思维动作** | 基于笔记内容执行深度分析、提炼摘要等智能任务 |
| **批量处理** | 对多篇笔记同时执行思维动作 |

## 最低配置

只需配置一个 **ASR Provider**（智谱 AI 或火山引擎豆包），即可使用语音转写功能。

配置 **LLM Provider** 后，AI 润色与思维动作功能全部解锁。
