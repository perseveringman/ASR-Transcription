# Provider 概览

插件使用两类 AI 服务：

| 类型 | 用途 | 支持的 Provider |
|---|---|---|
| **ASR（语音转文字）** | 录音转写、音频文件转写 | 智谱 AI、火山引擎豆包 |
| **LLM（大语言模型）** | AI 润色、思维动作 | OpenRouter、Gemini、DeepSeek、OpenAI、Anthropic、Minimax、智谱 GLM |

你至少需要配置一个 **ASR Provider** 才能使用转写功能，配置一个 **LLM Provider** 才能使用润色和思维动作。
