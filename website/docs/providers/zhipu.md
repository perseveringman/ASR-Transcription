# 智谱 AI（ASR）

## 适用场景

- 中英文混合识别
- 短音频（30 秒以内效果最佳）
- 对成本敏感的场景

## 配置步骤

1. 访问 [智谱 AI 开放平台](https://open.bigmodel.cn/) 注册并登录。
2. 在控制台获取 **API Key**。
3. 打开 Obsidian → 设置 → **ASR Transcription** → 转录服务商 → **智谱 AI**。
4. 粘贴 API Key 并保存。

## 注意事项

- 单段音频建议不超过 30 秒。
- 对于更长的音频，插件会自动分片处理，但处理时间会相应增加。
- 使用模型：**GLM-ASR-2512**
